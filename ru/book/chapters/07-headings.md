## Заголовки

В **Markdown** заголовки (**Headings**) используются для создания структуры документа. Они помогают разделить текст на разделы и подразделы, делая его более понятным и удобным для чтения.

Заголовки создаются с помощью символа `#`. Количество символов `#` определяет уровень заголовка: чем меньше символов используется, тем выше уровень заголовка.

### Заголовок первого уровня

Заголовок первого уровня создается с использованием одного символа `#`.

**Пример (Markdown):**

```markdown
# Заголовок первого уровня
```

**Результат (HTML):**

```html
<h1>Заголовок первого уровня</h1>
```

### Альтернативный способ создания заголовков первого и второго уровней

В Markdown существует альтернативный способ создания заголовков первого и второго уровней с использованием символов `=` и `-`.

**Пример (Markdown):**

```markdown
Заголовок первого уровня
========================

Заголовок второго уровня
------------------------
```

**Результат (HTML):**

```html
<h1>Заголовок первого уровня</h1>

<h2>Заголовок второго уровня</h2>
```

**Результат (Отображение):**

![](../images/heading-level-1-and-2.png)

При создании документов рекомендуется соблюдать логическую иерархию заголовков. Обычно документ начинается с заголовка первого уровня, затем используются заголовки второго уровня для основных разделов, заголовки третьего уровня для подразделов и, при необходимости, заголовки более низких уровней. Такая структура делает документ более удобным для чтения и навигации.
