## Список определений

Некоторые процессоры **Markdown** позволяют создавать **списки определений (Definition Lists)** для терминов и их объяснений. Это удобно, когда нужно описать технические понятия, команды или элементы программирования.

### Создание списка определений

Чтобы создать список определений, напишите термин на первой строке. На следующей строке введите двоеточие, затем пробел и само определение.

**Пример (Markdown):** 

```markdown
HTML
: Язык разметки, используемый для создания структуры веб-страниц.

CSS
: Язык стилей, применяемый для оформления и визуального отображения HTML-элементов.

JavaScript
: Язык программирования, который позволяет добавлять интерактивность на веб-страницы.

```

**Результат (HTML):** 

```html
<dl>
  <dt>HTML</dt>
  <dd>Язык разметки, используемый для создания структуры веб-страниц.</dd>

  <dt>CSS</dt>
  <dd>Язык стилей, применяемый для оформления и визуального отображения HTML-элементов.</dd>

  <dt>JavaScript</dt>
  <dd>Язык программирования, который позволяет добавлять интерактивность на веб-страницы.</dd>
</dl>

```
