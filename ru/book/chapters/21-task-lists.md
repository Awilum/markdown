## Чеклисты

В **Markdown**, **Task Lists** (или также известные как **чеклисты** и **списки задач**) предоставляют удобный способ создания списка элементов с возможностью отметки выполненных задач. В приложениях с **Markdown**, которые поддерживают **Task Lists**, флажки будут отображаться рядом с соответствующими элементами.

### Создание чеклистов

Для создания чеклиста используются дефисы `-` и квадратные скобки с пробелом `[ ]` перед элементами списка задач. Для отметки выполненной задачи вставляется символ `x` между квадратными скобками `[x]`.

**Пример (Markdown):** 

```markdown
- [ ] Невыполненная задача
- [x] Выполненная задача
- [ ] Еще одна невыполненная задача
```

**Результат (HTML):** 

```html
<ul>
  <li><input type="checkbox" disabled=""> Невыполненная задача</li>
  <li><input type="checkbox" disabled="" checked=""> Выполненная задача</li>
  <li><input type="checkbox" disabled=""> Еще одна невыполненная задача</li>
</ul>
```

### Использование вложенных списков

Чеклисты можно вкладывать внутрь обычных списков.

**Пример (Markdown):** 

```markdown
- [ ] Внешняя задача
  - [x] Вложенная выполненная задача
  - [ ] Вложенная невыполненная задача
- [ ] Еще одна внешняя задача
```

**Результат (HTML):** 

```html
<ul>
  <li>
    <input type="checkbox" disabled="" /> Внешняя задача
    <ul>
      <li><input type="checkbox" disabled="" checked="" /> Вложенная выполненная задача</li>
      <li><input type="checkbox" disabled="" /> Вложенная невыполненная задача</li>
    </ul>
  </li>
  <li><input type="checkbox" disabled="" /> Еще одна внешняя задача</li>
</ul>
```
