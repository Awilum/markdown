# Базовый синтаксис

В этом разделе вы изучите фундаментальные принципы языка разметки **Markdown**. Вы начнете с создания заголовков разных уровней для организации текста. После этого вы овладеете созданием параграфов и научитесь использовать переносы строк для повышения читаемости. Вы также познакомитесь элементами акцентирования, блоками цитат, списками и вставкой кода. Также в этом разделе будут рассмотрены горизонтальные линии, ссылки, вставка изображений и использование экранирующих символов для корректного отображения специальных символов.
